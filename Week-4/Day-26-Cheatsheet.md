**AGILE (10 terms):**
1. Agile - Iterative development methodology
2. Sprint - Fixed time period (2 weeks) for work
3. Stakeholder - Anyone interested in project
4. Scrum - Specific agile framework
5. Standup - Daily team sync meeting
6. Backlog - List of work to be done
7. User Story - Feature from user perspective
8. MVP - Minimum Viable Product
9. POC - Proof of Concept
10. Cross-functional - Team with diverse skills

**TESTING (10 terms):**
11. Regression Testing - Re-test after changes
12. Smoke Testing - Basic critical functionality tests
13. Test Coverage - % of code/features tested
14. Flaky Test - Inconsistent test results
15. Test Pyramid - Strategy: more unit, less UI tests
16. Shift-Left - Testing earlier in cycle
17. Parallel Execution - Running tests simultaneously
18. E2E Testing - Complete user workflow testing
19. API Testing - Testing interfaces/endpoints
20. Page Object Model (POM) - Design pattern for test code
